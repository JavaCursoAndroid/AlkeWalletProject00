Wallet Virtual - Proyecto Final SENCE

Aplicación de consola desarrollada en Java que permite:

- Registro de usuarios
- Inicio de sesión
- Depósitos
- Retiros
- Conversión de divisas
- Gestión de cuentas

Tecnologías utilizadas:
- Java
- Eclipse
- JUnit
- JDoc
- StarUML

Estructura:
- src/: código fuente
- docs/: documentación
- prototipo/: ejecutable
